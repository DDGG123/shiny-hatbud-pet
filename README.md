# Shiny Hatbud

Fan-made Codex pet spritesheet for a shiny Daermaodou / 大耳帽兜-inspired companion.

This repository stores the installable Codex pet package plus release snapshots and QA artifacts.

## Install

Copy the files in `pet/` into your Codex pet directory:

```text
%USERPROFILE%\.codex\pets\shiny-hatbud\
```

Expected package layout:

```text
shiny-hatbud/
  pet.json
  spritesheet.webp
```

## Current Version

`v1.0.0` is the first accepted Shiny Hatbud pet version.
